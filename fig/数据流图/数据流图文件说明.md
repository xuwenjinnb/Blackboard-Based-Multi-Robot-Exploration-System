# 数据流图文件说明

已按常见 DFD 样式重画：外部实体为矩形，加工为圆形，数据存储为上下两条横线，箭头只保留关键数据流。

| 文件 | 内容 |
| --- | --- |
| `01_一层数据流图.svg` | 类似参考图的一层 DFD：用户管理、实验配置、地图探索、回放分析四个子系统。 |
| `02_二层图_P1登录认证与权限控制.svg` | 用户管理子系统二层图：登录、角色分流、用户管理、用户信息表、登录会话表。 |
| `03_二层图_P3仿真配置.svg` | 实验配置子系统二层图：地图配置、障碍配置、小车配置、算法配置、初始化运行态。 |
| `04_二层图_P4-P7巡检运行核心闭环.svg` | 地图探索子系统二层图：小车扫描、前沿发现、任务分配、路径规划、路径执行。 |
| `05_二层图_P9回放记录与分析.svg` | 回放分析子系统二层图：记录运行帧、保存回放、查询列表、加载回放、播放分析。 |

报告中建议重点放 `01_一层数据流图.svg` 和 `04_二层图_P4-P7巡检运行核心闭环.svg`。如果老师要求“每个一层加工都要有二层展开”，再补充其余三张二层图。
