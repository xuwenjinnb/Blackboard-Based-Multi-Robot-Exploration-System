# 状态图文件说明

本目录保存“多小车协作巡检仿真系统状态图”。

| 文件 | 用途 |
| --- | --- |
| `01_多小车协作巡检仿真系统状态图.svg` | 正式矢量图，适合插入 Word、PPT 或继续在绘图工具中编辑 |
| `01_多小车协作巡检仿真系统状态图.png` | 由 SVG 导出的位图预览，适合快速查看或直接插入报告 |

状态图主要表达三部分内容：

1. 登录认证与角色分流：未登录、认证中、认证失败、管理员会话、运行员会话、分析员会话。
2. 系统运行状态：`STOPPED`、配置中、就绪、`RUNNING`、`PAUSED`、`RESETTING`、完成/停止。
3. 核心协作闭环：Robot 扫描和执行路径，Controller 分配任务，Navigator 规划路径，Redis 黑板保存共享状态并通过 WebSocket 回推前端。

图中保留了 `IDLE / SCANNING / MOVING / BLOCKED`、`PENDING / PLANNED / RUNNING / DONE / FAILED`、`SUCCESS / NO_PATH` 等关键状态，避免展开代码函数和算法公式，便于报告展示和答辩讲解。
